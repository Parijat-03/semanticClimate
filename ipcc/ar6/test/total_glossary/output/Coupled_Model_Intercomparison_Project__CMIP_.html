Coupled Model Intercomparison Project

A climate modelling activity from the World Climate Research Programme (WCRP) which coordinates and archives climate model simulations based on shared model inputs by modelling groups from around the world.

The (CMIP3) multi-model data set includes projections using Special Report on Emissions Scenarios (SRES) scenarios. The (CMIP5) data set includes projections using the Representative Concentration Pathways (RCP). The CMIP6 phase involves a suite of common model experiments as well as an ensemble of CMIP-endorsed Model Intercomparison Projects (MIPs).
